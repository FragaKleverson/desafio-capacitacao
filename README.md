# Desafio Final – Capacitação Git 

Este repositório foi criado como parte do **Desafio Final da capacitação de Git**, com o objetivo de praticar os principais conceitos de versionamento de código em um cenário próximo ao mundo real.

Aqui foram simuladas atividades comuns de trabalho em equipe, como criação de branches, commits, merge e resolução de conflitos.

---

##  Objetivo do Desafio

- Inicializar um repositório Git
- Criar e versionar arquivos `.txt`
- Trabalhar com branches
- Realizar push para repositório remoto
- Executar merge entre branches
- Simular e resolver conflitos de merge

---

##  Tecnologias Utilizadas

- Git
- GitHub
- Terminal (Git Bash)
- Visual Studio Code

---

##  Estrutura do Repositório

- `desafio_final_git.txt`  
  Arquivo principal criado na branch `master`, utilizado para simular commits e conflitos.

- `feature-agora-vai.txt`  
  Arquivo criado em uma branch de feature para simular desenvolvimento isolado.

- `README.md`  
  Documentação do projeto.

---

##  Branches Utilizadas

- **master**  
  Branch principal do projeto.

- **feature-agora-vai**  
  Branch criada para simular uma nova funcionalidade, posteriormente integrada à `master`.

---

##  Desafio Extra – Conflito de Merge

Foi simulado um conflito de merge ao editar a **mesma linha** de um arquivo em branches diferentes.

O conflito foi resolvido manualmente utilizando o Visual Studio Code, seguido de commit e push da resolução para o repositório remoto.

 Conflito resolvido com sucesso.

---

##  Resultado Final

- Fluxo de versionamento completo
- Histórico de commits organizado
- Branches criadas e integradas corretamente
- Repositório sincronizado com o GitHub

---

##  Aprendizados

- Importância do uso correto de branches
- Leitura e interpretação do `git status`
- Processo de merge e resolução de conflitos
- Boas práticas de versionamento
